# Nokia-Snake-Game
Classic Snake game in C++ using Raylib

A classic **Snake game** implemented in C++ using **Raylib**.  

This project demonstrates game development basics

## Features

- Play the game with **arrow keys**
  
- Snake grows when eating an apple
  
- Detects collisions with walls and tail
  
- Automatic reset on death
  
- Real-time score display
  
- Smooth graphics with Raylib
  
- Simple, clean, and fun game loop
